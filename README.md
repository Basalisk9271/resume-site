# Gabe Imlay Resume Site

A single-page, dark, tech-forward resume site built with plain HTML/CSS/JS. It highlights experience, skills, and credentials, and includes quick contact actions and downloadable assets.

## Preview
- Sections: Hero, About, Experience, Skills, Certifications, Projects, Contact
- Features: responsive layout, clipboard-copy email, gradient/glow styling, downloadable résumé and Security+ cert

## Getting Started
1) Open `index.html` in your browser (no build step required).
2) Update contact info and links in `index.html` if needed (email, LinkedIn, GitHub).
3) Replace assets as desired:
   - Résumé: `Gabriel Imlay_Resume.pdf`
   - Security+ cert: `ImlayG_Sec+.pdf`

## File Map
- `index.html` — page content and sections
- `styles.css` — layout, theme, and responsive styling
- `script.js` — clipboard copy helper for your email
- `Gabriel Imlay_Resume.pdf` — downloadable résumé
- `ImlayG_Sec+.pdf` — downloadable Security+ credential

## Deploy
Any static host works (GitHub Pages, Netlify, Vercel, S3). Point the host at the repository root; no build commands are required.
